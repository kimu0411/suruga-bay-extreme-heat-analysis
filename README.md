# Suruga Bay Extreme Heat Analysis

## 概要

このリポジトリは、大学の研究活動で実施した「駿河湾における極端高温現象の特性と外洋からの影響」の解析コードおよび解析結果をまとめたものです。

Pythonを用いてデータの前処理、しきい値の算出、極端高温イベントの抽出、可視化までを一貫して実装しました。

このプロジェクトは、新卒エンジニア職向けポートフォリオの成果物として公開しています。

---

## プロジェクト概要

### テーマ

駿河湾における極端高温現象の特性と外洋からの影響

### 目的

- 駿河湾における極端高温現象の特性を解析
- 外洋域との比較
- 発生日数・持続期間の評価
- Pythonによるデータ解析・可視化
